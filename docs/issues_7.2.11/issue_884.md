Describe the bug Temporary file does not delete itself when download is resumed with another link.
To Reproduce Steps to reproduce the behavior:
1. Start a download.
2. After it expires, refresh it with a new link and complete the download.
3. Check the temp file, the folder made will still be there and not deleted.
Expected behavior The temp file does not get auto-deleted and stays there.
