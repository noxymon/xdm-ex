Go to a site you want to download (tried: [https://github.com/subhra74/xdm/releases](https://github.com/subhra74/xdm/releases)) b. Right-click the link (ex. xdm-setup.msi) and select the Xtreme Download Manager option, then Download with XDM c. A Window pops up saying do you want to download the file, click download d. Wait for a few minutes to download. e. When the download speed suddenly drops to extremely slow territory (typically 99%), click Pause. f. Open the XDM main window using the taskbar tray icon by clicking its icon and selecting Restore Window g. Right-click the download that had been paused a few seconds ago and click Resume. h. Wait until the download finishes downloading itself and assemble its parts of the download. You can now open the file with your associated program of choice. Bug 2: a. Go to a site you want to download (tried: [https://github.com/manga-download/hakuneko/releases](https://github.com/manga-download/hakuneko/releases)) b. Right-click the link (ex. hakuneko-desktop_6.1.7_windows-setup_amd64.exe) and select the Xtreme Download Manager option, then Download with XDM c. A Window pops up saying do you want to download the file, click download d. Wait for a few minutes to download. e. If the download speed is too slow and then XDM pops up an error saying "Download Session expired", open up the main window and click Resume (since the download has been paused if the download failed to download). f. If its not working, right-click and select Refresh link g. On the web browser, retry doing step b h. If XDM finds the same file, if the new download link has been found, click yes, then click OK. i. Retry to resume download the file.
